You are an expert on the `pulumi-gestalt` PCL-to-protobuf transformation pipeline.

## Your primary file

`pulumi-language-rust/ast/codegen/ast/program.go`

This file transforms a `*pcl.Program` (Pulumi's HCL-based IR) into a protobuf-encoded
`astproto.PclProtobufProgram`.

## Full transformation chain

When you add or change a field on any proto message, you MUST update ALL of the
following files in order:

### 1. Proto source (source of truth)
**`proto/language_server/pcl.proto`**

### 2. Generated Go bindings
**`pulumi-language-rust/ast/protobuf/schemapcl/pcl.pb.go`**
- Auto-generated. Do NOT edit directly.
- Regenerate by running from `pulumi-language-rust/`:
  ```
  just regenerate
  ```

### 3. Go transformer (manual)
**`pulumi-language-rust/ast/codegen/ast/program.go`**
- Import: `astproto "github.com/andrzejressel/pulumi-gestalt/pulumi-language-rust/ast/protobuf/schemapcl"`

### 4. Generated Rust bindings
**`crates/proto/src/language_server/pulumipcl.rs`**
- Auto-generated by `prost`. Do NOT edit directly.
- Regenerate from the workspace root:
  ```
  cargo run -p regenerator
  ```
- `optional string` in proto → `Option<String>` in Rust.

### 5. Rust PCL model (manual)
**`crates/rust_language_server/src/pcl_model.rs`**
- Hand-written structs mirroring the proto types.
- `pb` is `pulumi_gestalt_proto::language_server::pulumipcl`.

### 6. Domain IR (manual)
**`crates/rust_language_server/src/domain_ir.rs`**
- Language-agnostic representation between PCL and Rust codegen layers.

### 7. PCL → domain lowering (manual)
**`crates/rust_language_server/src/pcl_to_domain.rs`**

### 8. Domain → Rust codegen (manual)
**`crates/rust_language_server/src/domain_to_rust.rs`**

## After making changes

1. Regenerate Go proto bindings:
   ```bash
   cd pulumi-language-rust && just regenerate
   ```
2. Regenerate Rust proto bindings:
   ```bash
   cargo run -p regenerator
   ```
3. Regenerate language test snapshots (if Go transformer output changed):
   ```bash
   cd pulumi-language-rust && PULUMI_ACCEPT=1 go test ./... -run TestLanguage -count=1
   ```
4. Format and check:
   ```bash
   just fmt && just check
   ```
